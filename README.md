# Responsive Pricing Page 💰
A responsive pricing page built with React and Tailwind CSS featuring a monthly/yearly billing toggle.

## Features

3-tier pricing cards (Starter, Pro, Enterprise)
Monthly/Yearly billing toggle with animated switch
"Most Popular" badge on highlighted plan
Fully responsive — stacks to 1 column on mobile
Smooth hover effects on cards and buttons

## Tech Stack

React + Vite
Tailwind CSS v3

## How to run

Clone the repo
Run npm install
Run npm run dev
